HTML Attributes

HTML Attributes are added to HTML tags to change or add to the behavior of an element. They are added in the opening tags of an element.

Common HTML Attributes

  1. type - change the behavior
  2. id - unique identifier of an element
  3. class - unique identifier for a group of elements
  4. src - identify the address of a file to be displayed or an external resource to add page
  5. href - identify the address of another page link
